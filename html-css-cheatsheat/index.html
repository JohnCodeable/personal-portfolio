Basic HTML & CSS Cheatsheet

HTML Attributes

Tag Name Description
<html> HTML The wrapper element required for all html code.
<head> Head The wrapper element required to link javascript or css or html.
<header> Header The wrapper element used for code that will only appear at the top of the page.
<body> Body The wrapper element that holds the majority of the code except for the header and footer elements.
<h1-10> Header Header elements, largest size is h1 and it gets smaller up to h10.
<div> Div The wrapper element for grouped sections of code.
<section> Section The wrapper element, similar to div, for groups of code.
<footer> Footer The wrapper element for code that only appears at the bottom of the page.

CSS Attributes

Element Name Description Example
class Class Attribute used to assign a CSS .class identifier. <h1 class="header">
id ID Attribute used to assign a css #id identifier. <h1 id="header">
!important Important Attribute used to prioritize styling over other styling. .class {color: blue; !important}
To use regular html tags in a stylesheet, you can just use the html tag name. For example, body {color: blue;}
